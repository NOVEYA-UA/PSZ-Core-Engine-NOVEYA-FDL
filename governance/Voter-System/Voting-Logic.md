# 🗳️ Протокол Прямого Волеизъявления

## Механика:
* **Инициация:** Предложение выносится в раздел `GitHub Discussions`.
* **Голосование:** Осуществляется через реакцию на коммит или специализированные Issues.
* **Валидация:** Голос учитывается только при наличии у актора СИП > 60.
* **Результат:** При достижении кворума решение автоматически коммитится в реестр как «Принято громадой».
