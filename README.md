🏛️ ПАСПОРТ ЭКОСИСТЕМЫ «Σ-FDL • НОВЕЯ» (v1.0)
1. ОБЩЕЕ ОПИСАНИЕ (ОНТОЛОГИЯ)
НОВЕЯ — это не программное обеспечение, а Генеративная Биосистема. Она предназначена для автоматизированного перевода социальных инициатив (жалоб, предложений) в неизменяемые юридические акты через цикл «Тезис — Антитезис — Синтез».
________________________________________
2. АРХИТЕКТУРА УЗЛОВ (ГДЕ ЧТО ЛЕЖИТ)
Узел	Название	Инструмент	Роль в Системе
Узел 01	LEDGER (КУБ v1)	Google Sheets	Регистрация «Сырой энергии». Присвоение ID.
Узел 02	CORE (Ядро)	Python (app.py)	«Мельник». Перекачивает данные между узлами.
Узел 03	STAFF (Штаб)	Notion DB	Собор смыслов. Навигационная карта векторов.
Узел 04	VAULT (Хранилище)	GitHub	Бессмертный контур. Фиксация кода и актов.
Узел 05	VOICE (Голос)	Telegram Bot	Оперативная отчетность Архитектору.
________________________________________
3. ПОЛЬЗОВАТЕЛЬСКАЯ ИНСТРУКЦИЯ (ПУТЬ ЗЕРНА)
Этап А: Инициация (Посев)
Пользователь заполняет форму на портале. Данные падают в Google Таблицу.
•	Важно: Каждая запись получает уникальный номер (напр. N-001). Это «имя» вашего намерения.
Этап Б: Трансформация (Рост)
Скрипт app.py на локальном ПК (или сервере) подхватывает запись.
1.	Проверяет, нет ли дубликатов в Notion.
2.	Если запись новая — создает страницу в «Соборе» (Notion).
3.	Шлет уведомление в Telegram: «Материализация Σ-FDL: Зафиксировано».
Этап В: Синтез (Жатва)
Юристы и аналитики в Asana превращают этот тезис в правовой акт. Финальный результат «заливается» в GitHub (директория /governance), становясь частью неизменяемой истории.
________________________________________
4. ИНСТРУКЦИЯ ДЛЯ ТЕХНИЧЕСКОГО КОНТРОЛЯ (АДМИН)
Как запустить «Двигатель»:
1.	Открыть терминал PowerShell.
2.	Перейти в папку проекта: cd C:\Metatron_Project\Metatron-8.
3.	Запустить Ядро: py app.py.
4.	Статус «Меридиан закрыт»: Означает, что нужно в Google Таблице нажать Deploy -> Anyone. Как только шлюз откроется, надпись сменится на «Ожидание сигналов».
Как обновить Хранилище (GitHub):
Если ты внес изменения в методологию или код локально, используй «Команду Силы»:
PowerShell
git add --all
git commit -m "Синхронизация Снопа"
git push origin main
